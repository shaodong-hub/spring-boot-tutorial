== 面向切面编程 ⭐️
:doctype: book
:sourcedir: src/docs/asciidoc
:numbered:

AOP为Aspect Oriented Programming的缩写，意为：面向切面编程，通过预编译方式和运行期间动态代理实现程序功能的统一维护的一种技术(装饰器模式区别).

AOP主要是提供另外一种编程思路，可以把类似的行为抽离出来统一处理.

例如:Aspect(`@Aspect`), 事务(`@Transaction`), 异步(`@Async`), 重试机制(`@Retry`)
