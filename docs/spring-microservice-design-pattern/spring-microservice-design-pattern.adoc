一些语义
findXxx，有可能找到也可能找不到。所以返回值可能为 null，可以返回为 Optional；
generateXxxx ，每次调用该方法返回的数据都不一样；
getXxx，每次调用该方法返回的数据都一样；
fillXxx，对传入的集合或者对象进行填充数据；
